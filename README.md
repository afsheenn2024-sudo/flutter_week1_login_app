# Flutter Week 1 – Login Application

This Flutter application was developed as part of the Flutter Internship Week 1 assignment. The project demonstrates the fundamentals of Flutter by creating a simple login interface with form validation and navigation between screens.

## Features

* User login interface
* Email and password input fields
* Form validation
* Navigation from the Login Screen to the Home Screen
* Simple and user-friendly interface

## Technologies Used

* Flutter
* Dart

## Project Structure

```text
lib/
├── main.dart
└── screens/
    ├── login_screen.dart
    └── home_screen.dart
```

## Installation & Setup

### Clone the repository

```bash
git clone https://github.com/afsheenn2024-sudo/flutter_week1_login_app/tree/main
```

### Open the project

```bash
cd flutter_week1_login_app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```

## How to Use

1. Launch the application.
2. Enter an email address.
3. Enter a password.
4. Complete the required validation.
5. Tap the **Login** button.
6. After successful validation, the application navigates to the Home Screen.
